init(app, paFreeFormId)
입력용 폼레이아웃 컨트롤들에 대해 초기화 로직을 수행한다.
폼레이아웃의 class는 form-box 지정 필수
1. 데이터 여부에 따른 비활성화 처리 (load, filter 이벤트)
(데이터가 없으면 입력 안됨 처리)
